Live2D Cubism SDK Setup

Live2D Cubism Core is required to display Live2D models.
Due to licensing, it must be downloaded from the official site.

1
Download the Cubism SDK for Web from the official page (free, requires license agreement)
2
Drop the downloaded .zip file (or live2dcubismcore.min.js) below
3
ELINO will extract and place the required files automatically
